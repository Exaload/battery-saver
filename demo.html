ðŸ“Š Battery Saving Modes

Normal Mode (>50% battery or charging)

  • All animations and transitions work normally
  • Full GPU acceleration enabled
  • Parallax and complex effects active

Medium Mode (30-50% battery)

  • Animation duration reduced to 0.5s
  • Transition duration reduced to 0.3s
  • Parallax effects disabled
  • Frame rate optimized

Low Mode (15-30% battery)

  • Animation duration reduced to 0.2s
  • Transition duration reduced to 0.15s
  • Shadows and blur effects removed
  • Video autoplay disabled
  • Frame rate capped at 30fps

Critical Mode (<15% battery)

  • All animations completely disabled
  • All transitions completely disabled
  • All shadows, blurs, and gradients removed
  • Maximum power saving mode

ðŸš€ Quick Start

To use Battery Saver in your project:

1. Include the files:

<link rel="stylesheet" href="battery-saver.css">
<script src="battery-saver.js"></script>

2. Auto-initialize:

<html data-battery-saver-auto>

3. Or manually initialize:

const batterySaver = new BatterySaver();

// Listen to events
batterySaver.on('modeChange', (data) => {
  console.log('Mode changed:', data);
});

batterySaver.on('batteryChange', (data) => {
  console.log('Battery updated:', data);
});
